## 📌 Overview  
SubAudit Pro is a **subscription tracking and expense analysis web application** that helps users manage all their subscriptions in one place.  
It provides insights into spending patterns and highlights unused subscriptions to reduce financial waste.

---

## ✨ Key Features  

✔️ Centralized subscription tracking  
✔️ Real-time expense monitoring  
✔️ 📊 Visual dashboard with charts  
✔️ ⚠️ Detect unused subscriptions  
✔️ 📅 Renewal date reminders  
✔️ 💸 Expense optimization insights
